# appium-boilerplate

To set up
```
npm install
```

To see if eslint has been followed
```
npm run lint
```

To fix all formatting errors
```
npm run lint:fix
```

To run iOS tests
```
npm run ios.app
```

To run Android tests
```
npm run android.app
```
